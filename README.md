# 题库编辑器 (Question Generator)

一款基于 PyQt5 的跨平台题库编辑工具，支持多种题型、图片插入以及按科目/年级组织题库。

## 功能特点

- **多种题型**：选择题、填空题、简答题、拖拽配对、连线题
- **图片支持**：可为题目配图（jpg、png、bmp、gif、tiff），图片大小限制 10MB
- **配对题型**：拖拽配对和连线题支持动态添加/删除配对项，使用表格编辑左右对应关系
- **自动保存**：编辑题目后自动保存至当前题库文件，无需手动点击保存
- **文件管理**：新建、打开、保存、另存为 JSON 格式的题库文件
- **目录组织**：题库按科目/年级存储在系统应用数据目录下（Windows: `%APPDATA%\题库编辑器`，macOS: `~/Library/Application Support/题库编辑器`，Linux: `~/.config/题库编辑器`）
- **首次运行配置**：启动时自动弹出科目与年级选择对话框，生成默认题库路径
- **异常保护**：程序崩溃时自动生成错误日志文件，便于排查问题

## 环境依赖

- Python 3.6 或更高版本
- 依赖库：
  - `PyQt5`
  - `Pillow` (PIL)
